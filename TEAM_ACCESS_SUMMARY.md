## Getting Started

```bash
# Clone the repository
git clone https://github.com/candlefish-ai/candlefish-ai.git

# Enter the directory
cd candlefish-ai

# Install dependencies (we use pnpm)
npm install -g pnpm
pnpm install

# Run a specific project
cd brand/website  # or projects/paintbox, etc.
npm run dev
```
