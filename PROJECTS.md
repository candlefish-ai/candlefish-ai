## ⚡ Quick Commands

### Navigation
```bash
cf          # Go to root
cfweb       # Website project
cfpaint     # Paintbox project
cffogg      # Fogg calendar
cfprom      # PromoterOS
cfbrew      # Brewkit
cfcrown     # Crown Trophy
cfbart      # BART analytics
```

### Project Actions
```bash
cfbuild     # Build current project
cftest      # Run tests
cfdeploy    # Deploy to production
```
