### Deployment Steps
1. **Install dependencies** (if not already done):
   ```bash
   cd /Users/patricksmith/candlefish-ai
   pnpm install
   ```

2. **Build the website**:
   ```bash
   cd apps/website
   pnpm build
   ```

3. **Deploy to Netlify**:
   ```bash
   pnpm deploy
   ```
   Or manually:
   ```bash
   netlify deploy --prod --dir=dist
   ```
